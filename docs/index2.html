The Phantom Trees Growth - Unity scene

Assigment II

As the days passes the phantom trees grow and shine at night...


You can try out the web version here or download the PC version here.
The web version have various things stripped out (mainly dynamic shadows/effects).
You can use the controls "W" "A" "S" "D" "Q" "E" to move forward, backward, left, right, up and down respectively. Pressing "SPACE" you can toggle between skip the animation and reset the animation.
It's a CPU intensive task as the trees meshes are generated every frame for the animation.
We are using about 200 million triangles in the scene, so you may notice a frame drop when the trees are growing.
We could run it fine on a old i7 3770, so if you have a better processor and enough system and GPU RAM to store the triangles you should be fine.
We have plans to reduce the triangles count by simplifying the meshes given the viewing distance and improve performance by multi-threading the mesh generation or using compute shader.
